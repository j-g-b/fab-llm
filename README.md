# fab-llm
Code to reproduce the results and figures from article "Incorporating Large Language Model-Derived Information Into Hypothesis Testing For Genomics"

# Instructions

Alongside the files downloaded from this repository, create the following three directories:

- `data`
- `figures`
- `results`

Download the .zip file from this [figshare location](https://doi.org/10.6084/m9.figshare.32434986) and unzip the contents into the `data` directory. Then run the desired .R script.
